# Prompt Template: Bug Fix

Use this template when reporting a bug to AI:

```
## Bug Report

### Error Message
{Paste exact error from console}

### Steps to Reproduce
1. Go to {page}
2. Click {action}
3. See error

### Expected Behavior
{What should happen}

### Actual Behavior
{What actually happens}

### Environment
- Browser: Chrome/Edge/Safari
- Device: PC/Tablet/Mobile
- URL: http://localhost:3001/...

### Screenshots
{Paste screenshot if available}

### Related Files
- Frontend: src/...
- Backend: apps/...
- Database: table name
```
