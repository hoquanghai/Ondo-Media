# Prompt Template: New Feature

Use this template when asking AI to implement a new feature:

```
## Feature: {Feature Name}

### Description
{What the feature does}

### User Stories
- As a {role}, I want to {action} so that {benefit}

### API Endpoints
- GET /api/v1/...
- POST /api/v1/...

### Database Changes
- New table: {table_name}
- New columns: {table}.{column}

### UI Requirements
- Page route: /...
- Components needed: ...
- Responsive: PC/Tablet/Mobile

### Acceptance Criteria
- [ ] ...
- [ ] ...

### References
- Design doc: docs/features/{name}/spec.md
- Figma: {link if available}
```
